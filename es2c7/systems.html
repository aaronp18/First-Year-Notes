Oscillators & State Space Systems

Modal Analysis

Oscillators are coupled mass/spring, pendulums, etc systems, which can be analysed using modal analysis:

  • Start with a complex coupled system
  • Use spectral decomposition to diagonalise the system into simpler uncoupled systems
  • Solve for each system

Single Degree of Freedom Oscillators

Mass-Spring

The equation of motion is:

dt2d2z(t)​=−kz(t)

where k is the normalised stiffness, k=kˉ/m. Assuming an oscillatory solution:

z(t)=Acosω0​t+Bsinω0​t z˙(t)=−ω0​Asinω0​t+ω0​Bcosω0​t z¨(t)=−ω02​Acosω0​t−ω02​Bsinω0​t

Solving for ω0​ by substituting back in gives ω0​=k​.

Setting z(0)=0 and z˙(0)=v0​:

z(t)=asin(ω0​t)a=ω0​v0​​

This system oscillates at a single frequency, ω0​=kˉ/m​.

Pendulum

The equation of motion for a pendulum in the tangential direction is:

mlθ¨=−mgsinθ θ¨(t)=−kθ(t)(for smallθ)

Where k=g/l

  • The system oscillates at the frequency ω0​=k​
  • This system has the same form, and therefore solution, as the mass-spring.
  • The frequency depends only on the length, not the mass, a property unique to pendulums.

Multiple Degrees of Freedom

This single degree of freedom can be generalised to a 2nd order n-degree of freedom system:

dt2d2y​=−Ky(t)

K=⎣⎡​k11​k21​⋮kn1​​k12​k22​⋮kn2​​……⋱…​k1n​k2n​⋮knn​​⎦⎤​y=⎣⎡​y1​y2​⋮yn​​⎦⎤​

  • K is an n×n matrix
  • y is an n-dimensional column vector

The goal is to find frequencies ω such that the solution y(t) can be expressed as harmonic functions of sin(ωt). This is done by spectral decomposition:

K=VΛV−1 dt2d2y​=−VΛV−1y(t) dt2d2V−1y​=−V−1VΛV−1y(t)

Introduce a new variable z(t)=V−1y(t), so that for z(t):

dt2d2z​=−Λz(t)

Λ=⎣⎡​λ1​0⋮0​0λ2​⋮0​……⋱…​00⋮λn​​⎦⎤​

This equation involving a diagonal matrix can then be decomposed to n uncoupled scalar equations (the normal modes) for each scalar zi​ in z:

dt2d2zi​​=−λi​z(t)

This is a single degree of freedom scalar equation, as the previous two examples, thus:

  • λi​=ωi2​
  • z(t)=asin(ωt)

The solution of the 2nd order n-DoF system is defined by a superposition of the normal modes zi​(t)

  • λi​=ωi2​ is an eigenvalue of K
    • ωi​ is the frequency of the normal mode
  • vi​ is an eigenvector of K
    • Specifies the shape of the normal mode

Example 1

Conside a system of two coupled masses:

  • Two masses m1​ and m2​
  • Two displacements y=[y1​y2​​]
    • The variable to solve for
  • Three springs k1​, k2​, k3​

Two equations of motion, one for each mass:

m1​dt2d2y1​​=−k1​y1​−k3​(y1​−y2​) m2​dt2d2y2​​=−k2​y2​+k3​(y1​−y2​)

dt2d2y1​​=m1​−(k1​+k3​)y1​+k3​y2​​ dt2d2y2​​=m2​k3​y1​−(k3​+k3​)y2​​

dt2d2y​=−[m1​k1​+k3​​m2​−k3​​​m1​−k3​​m2​k2​+k3​​​][y1​y2​​]=−Ky

Let k1​=k2​=2, and k3​=m1​=m2​=1:

K=[3−1​−13​]

To solve the system, need to compute the eigenvalues and eigenvectors of K, and hence the normal modes. Starting with the eigenvalues:

∣∣​3−λ−1​−13−λ​∣∣​=0

λ2−6λ+8=0 λ1​=2λ2​=4

Hence the two natural frequencies of oscillation are ω1​=2​ and ω2​=2. Now for the eigenvectors:

(A−λi​)v=0

The first mode λ1​=2, v(1)=[11​], ω1​=2​ implies that both bodies move in unison at the frequency of the mode f = \frac{1}{\sqrt{2\pi}Hz. The spring between the two masses does not stretch or contract.

The second mode λ1​=4, v(2)=[1−1​], ω2​=2 implies that both bodies move in opposition at the frequency of the mode f=π1​Hz, with the connecting spring stretching and contracting.

State Space Linear Systems

Consider a second order linear ODE of the form y¨​(t)+ay˙​(t)+by(t)=0. Two variables are needed to uniquely specify the state of the system at any moment in time, the displacement x1​=y(t), and the velocity x2​=y˙​(t). The system can be rewritten in terms of these:

x1​˙​=y˙​=x2​ x2​˙​=y¨​=−ay˙​−by=−ax2​−bx1​

[x1​˙​x2​˙​​]=[0−b​1−a​][x1​x2​​]

This has replaced a 2nd order scalar equation with a two-state 1st order matrix equation. This concept can be generalised to express an nth order linear ODE as an n-state first order linear matrix ODE:

⎣⎡​x1​˙​x2​˙​⋮xn​˙​​⎦⎤​=⎣⎡​00⋮0−an​​10⋮0−an−1​​01⋮0−an−2​​……⋱……​00⋮1−a1​​⎦⎤​⎣⎡​x1​x2​⋮xn​​⎦⎤​

Where the state vector x(t)=[x1​​x2​​…​xn​​]′

  • x1​=y
  • x2​=y′=x1​˙​
  • x3​=y′′=x2​˙​
  • xn​=y(n)=xn−1​˙​

Now to work out how to solve it. In the scalar case, the solution to x˙(t)=ax(t) with x(0)=0 has the form

x(t)=eatx0​=x0​(1+at+2!(at)2​+3!(at)3​+…)

The sign of a determines the stability of the system:

  • a is negative: the system decays exponentially and is stable
  • a is zero: nothing ever happens
  • a is positive: the system rises exponentially and is unstable

The matrix case has the same solution:

x(t)=eAtx0​=x0​(In​+At+2!(At)2​+3!(At)3​+…)

The task is then to compute the matrix exponential, and characterise the dynamics of the solution using the matrix A.

Suppose A has the spectral decomposition A=VΛV−1:

x˙(t)=Ax(t)=VΛV−1x(t) x˙(t)=Ax(t)=ΛV−1x(t)

Defining z(t)=V−1x(t) again:

z˙(t)=Λz(t)

z(t)=⎣⎡​eλ1​t0⋮0​0eλ2​t⋮0​……⋱…​00⋮eλn​t​⎦⎤​z(0)

Since Λ is diagonal, this is now a set of uncoupled equations:

z1​˙​(t)=λ1​z1​(t)⟹z1​(t)=eλ1​tz1​(0) z2​˙​(t)=λ1​z2​(t)⟹z2​(t)=eλ1​tz2​(0) zn​˙​(t)=λ1​zn​(t)⟹zn​(t)=eλ1​tzn​(0)

z1​(t),z2​(t),…,zn​(t) are the individual modes of the solution x(t) and are defined by the eigenvalues λ1​,λ2​,…,λn​ alone. The solution x(t) is given by:

x(t)=Vz(t)=Vdiag{eλ1​t,eλ2​t,…,eλn​t}V−1x0​

  • The solution is a linear combination of the terms eλ1​t,eλ2​t,…,eλn​t
  • Hence, behaviour is defined by the eigenvalues
  • The system is stable if all eigenvalues are negative
  • If at least one is positive, the system is unstable

Example

Consider an elementary RLC circuit with all components in series, with a non-zero initial charge q0​ on the capacitor. The instantaneous charge q(t) in the circuit is described by a linear state space differential equation where x(t)=[q(t)q˙​(t)​] and x0​=[q0​0​]. Suppose:

  • q0​=6
  • R=7
  • L=1

The state space equation for the system in the form x˙(t)=Aq(t) is:

[x1​˙​(t)x2​˙​​]=[0−10​1−7​][q(t)q˙​(t)​]

The eigenvalues and eigenvectors of A are:

λ1​=−2v(1)=[1−2​]

The solution is given by x(t)=eAtx0​, and the matrix exponential term eAt=Vdiag{eλ1​t,eλ2​t,…,eλn​t}V−1:

x(t)=[1−2​1−5​][e−2t0​0e−5t​](3−1​)[−52​−11​][60​]=[−10e−2t−4e−5t−20e−2t+20e−5t​]

Also, since both eigenvalues <0, the system is stable.
