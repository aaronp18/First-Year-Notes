Implicit Differentiation

When differentiating a function of one variable with respect to another (ie dxdy​f(y)), simply differentiate with respect to y, then multiply by dxdy​.

For example, find dxdy​ where x2y3−x2+3y−3=0. First, using the product rule to differentiate the first term: u=x2u′=2x v=y3v′=3y2dxdy​

The equation with all terms differentiated: 2xy3+3x2y2dxdy​−2x+3ydxdy​=0

Rearranging to get in terms of dxdy​: 3x2y2dxdy​+3ydxdy​=2x−2xy3 dxdy​=3x2y2+32x−2xy3​

Differentials

Differentials describe small changes to values/functions y=f(x) dxdy​=f′(x) dy=f′(x)dx

Recall that dxdy​≈δxδy​. This means this can be rewritten: δy≈f′(x)δx

Dividing both sides by y=f(x): yδy​=f(x)xf′(x)​xδx​

yδy​ represents a relative change in y, and xδx​ represents a relative change in x. This can be used to give approximations of how one quantity changes based upon another.

For example, given the mass of a sphere M=34​ρπr3, where ρ is the material density, estimate the change in mass when the radius is increased by 2%. M=34​ρπr3 drdM​=4ρπr2 δm=4ρπr2δr

Dividing both sides by the original formula: MδM​=34​ρπr34ρπr2δr​=3rδr​

rδr​ represents a relative change in radius, so when r increases by 2%, rδr​=0.02 MδM​=3×0.02=0.06

Meaning the mass increases by 6%.

Hyperbolic Functions

Hyperbolic functions have similar identities to circular trig functions. They're the same, except anywhere there is a product of two sinhs, the term should be negated. Hyperbolic functions can also be defined in terms of exponential functions, making them easy to differentiate.

y=coshx=21​(ex+e−x) dxdy​=21​(ex−e−x)=sinhx

All the derivatives of hyperbolic functions are given in the formula book.

Partial Differentiation

For a function of two variables z=f(x,y) there are two gradients at the point z, one in x and one in y. To find the gradient in the x direction, differentiate f(x,y) treating y as a constant. To find the gradient in the y direction, differentiate f(x,y) treating x as a constant. These are the two partial derivatives of the function, ∂x∂z​ and ∂y∂z​.

For example, for a function z=4x3+5y7x: ∂x∂z​=12x2+5y7 ∂y∂z​=35y6x

Implicit Partial Differentiation

When a function of several variables is given and a partial derivative is required, differentiate the numerator of the partial derivative implicitly with respect to the denominator, and treat the third variable as constant. For example, find ∂y∂z​ given z2=x2+y2:

∂y∂​z2=∂y∂​(x2+y2)

2z∂y∂z​=2y ∂y∂z​=zy​

Another example, find ∂x∂z​ given zcosz=x2y3+z ∂x∂​(zcosz)=∂x∂​(x2y3+z) ∂x∂z​cosz−zsinz∂x∂z​=2xy3+∂x∂z​ ∂x∂z​(cosz−zsinz−1)=2xy3 ∂x∂z​=cosz−zsinz−12xy3​

Higher Order Partial Derivatives

Three 2nd order derivatives for functions of 2 variables. For z=f(x,y): ∂x2∂2z​=∂x∂​(∂x∂z​) ∂y2∂2z​=∂y∂​(∂y∂z​) ∂x∂y∂2z​=∂x∂​(∂y∂z​)=∂y∂​(∂x∂z​)=∂y∂x∂2z​

Note how for the last one, the order is interchangable as it yields the same result.

Chain Rule

The chain rule for a function w(x,y), where x and y are functions of a parameter t: dtdw​=∂x∂w​dtdx​+∂y∂w​dtdy​

Total Differential

The total differential represents the total height gain or lost when moving along the function described by z=f(x,y) dz=∂x∂f​dx+∂y∂f​dy

Contour Plots

Along a line of a contour plot, the total differential is zero: the height doesn't change. This allows dxdy​ to be found dh=∂x∂h​dx+∂y∂h​dy=0 dxdy​=∂h/∂y∂h/∂x​
